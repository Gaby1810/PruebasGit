¿Qué es Git?

Git fue creado pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. Git proporciona herramientas para desarrollar trabajo en equipo de manera inteligente y rápida.

El control de versiones permite llevar un historial de cambios en un proyecto, facilitando la colaboración entre varias personas y evitando la pérdida de información durante el desarrollo de software o páginas web.

Este cambio fue realizado para practicar el uso de Git, GitHub y la gestión de versiones mediante repositorios locales y remotos.
